# 🥋 Chuck Norris Joke Generator

A fun web application that fetches random Chuck Norris jokes from the [Chuck Norris API](https://api.chucknorris.io/) and displays them in an interactive card layout.

## ✨ Features

- **3 Random Jokes** - Fetches three different Chuck Norris jokes simultaneously
- **Refresh All** - Click the "Give Me New Jokes" button to refresh all three jokes
- **Individual Refresh** - Click on any joke card to refresh just that specific joke
- **Responsive Layout** - Jokes display side-by-side on desktop and stack on mobile
- **Dark Theme** - Sleek black and dark gray color scheme

- **HTML5** - Structure and layout
- **CSS3** - Styling, flexbox layout, dark theme
- **JavaScript (ES6+)** - Async/await, Fetch API, DOM manipulation
- **Chuck Norris API** - Free joke database API
